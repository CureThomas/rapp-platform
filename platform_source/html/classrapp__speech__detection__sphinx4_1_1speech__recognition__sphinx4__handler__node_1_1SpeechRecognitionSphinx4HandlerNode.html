Detailed Description

Maintains Sphinx instances to perform speech recognition.

Maintains a number of child processes to perform speech recognition utilizing Sphinx4 (rapp_speech_detection_sphinx4.speech_recognition_sphinx4.SpeechRecognitionSphinx4). Provides ros services and handles the requests according to the child processes' status.
