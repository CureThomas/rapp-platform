Detailed Description

Contains the Sphinx subprocess and is responsible for configuring Sphinx and performing the recognition request.

Initializes a Sphinx.java subprocess and creates an IPC using sockets. It is responsible for interacting with Sphinx via the socket to send configuration params/instructions and initialize a recognition procedure.
