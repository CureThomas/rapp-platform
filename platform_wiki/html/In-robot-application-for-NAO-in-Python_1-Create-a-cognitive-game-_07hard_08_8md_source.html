7 ## Preparation steps
8 
9 For this tutorial we will use the following tools:
10 - A real NAO robot
11 - The [rapp-robots-api](https://github.com/rapp-project/rapp-robots-api) for commanding NAO Robot.
12 - The [Python rapp-platform-api](https://github.com/rapp-project/rapp-api/tree/master/python) as we will need to call several RAPP Platform Services.
13 
14 Of course the standard prerequisites are an editor and a terminal.
15 
16 #### RAPP Robots API libraries setup
17 
18 The first step is to clone the appropriate GitHub repository in your PC:
19 
20 ```bash
21 $ mkdir ~/rapp_nao
22 $ cd ~/rapp_nao
23 $ git clone https://github.com/rapp-project/rapp-robyyots-api.git
24 ```
25 
26 The next step is to transfer the RAPP Python libraries to the NAO robot. This will be done via ```scp```, assuming that the NAO robot's IP is ```192.168.0.101``` and username and password are ```nao```:
27 
28 ```bash
29 $ cd ~/rapp_nao/
30 $ tar -zcvf rapp_robots_api.tar.gz rapp-robots-api/
31 $ scp rapp_robots_api.tar.gz nao@192.168.0.101:/home/nao
32 ```
33 
34 Now connect in NAO via ssh by ```ssh nao@192.168.0.101``` giving ```nao``` as password. Then untar the API:
35 
36 ```bash
37 $ tar -xvf rapp_robots_api.tar.gz
38 $ rm rapp_robots_api.tar.gz
39 ```
40 
41 The next step is to update the ```PYTHONPATH``` variable. Since NAO has Gentoo as OS, we will modify the ```bash_profile``` file:
42 
43 ```bash
44 $ echo 'export PYTHONPATH=$PYTHONPATH:/home/nao/rapp-robots-api/python/abstract_classes' >> /home/nao/.bash_profile
45 $ echo 'export PYTHONPATH=$PYTHONPATH:/home/nao/rapp-robots-api/python/implementations/nao_v4_naoqi2.1.4' >> /home/nao/.bash_profile
46 $ source ~/.bash_profile
47 ```
48 
49 The last step to configure the ```rapp-robots-api``` is to declare the NAO IP. Since the robot API is in-robot, the IP must be localhost: ```127.0.0.1```.
50 
51 The IP must be declared in the first line of [this](https://github.com/rapp-project/rapp-robots-api/blob/master/python/implementations/nao_v4_naoqi2.1.4/nao_connectivity) file, thus the ```nao_connectivity``` file located under ```/home/nao/rapp-robots-api/python/implementations/nao_v4_naoqi2.1.4/nao_connectivity``` should contain:
52 
53 ```
54 127.0.0.1
55 9559
56 ```

92 ##Writing the Cognitive Exercises RApp
93 
94 We will work on the host machine while implementing the RApp and then we are going to transfer the application source files to the NAO Robot and execute.
95 
96 Let's create a package for the application:
97 
98 ```bash
99 $ mkdir -p ~/rapp_nao/rapps/cognitive_exercises
100 $ cd ~/rapp_nao/rapps/cognitive_exercises
101 $ mkdir rapps && cd rapps
102 $ touch cognitive_exercises.py
103 $ chmod +x cognitive_exercises.py
104 ```
105 
106 Import the required components into the code:
107 
108 ```python
109 #!/usr/bin/env python
110 
111 # Import the RAPP Robot API
112 from rapp_robot_api import RappRobot
113 # Create an object in order to call the desired functions
114 rh = RappRobot()
115 
116 from RappCloud import RappPlatformService
117 from RappCloud.CloudMsgs import (
118  CognitiveExerciseSelect,
119  CognitiveRecordPerformance,
120  SpeechRecognitionSphinx,
121  SetNoiseProfile)
122 ```
