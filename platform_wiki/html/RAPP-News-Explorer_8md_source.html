1 Rapp News Explorer allows users to search for news articles employing various
2 news search engine APIs.
3 The user can specify the desired news search engine, topics (i.e. sports,
4 economy, headlines, etc.), desired keywords, language/region (can be specified
5 employing
6 [RAPP Geolocator](https://github.com/rapp-project/rapp-platform/wiki/RAPP-Geolocator)),
7 etc. The node returns a list of related articles.
8 The availability of news_explorer services that rely on third party APIs such
9 as Google News are restricted according to the APIs' rules and limitations.
10 Thus, service call failures may exist.
11 
12 Currently supported News Engines:
13 * Google News ([API](https://developers.google.com/news-search/v1/devguide)) (discontinued by Google)
14 * [EventRegistry](http://eventregistry.org/) ([GitHub wiki](https://github.com/gregorleban/EventRegistry/wiki))
15 
16 EventRegistry does not require login, however it is subjected to [daily access restrictions.](https://github.com/gregorleban/EventRegistry/wiki/Daily-access-restrictions) If you have an EventRegistry account, you should provide the credentials in the file `${HOME}/.config/rapp_platform/api_keys/event_registry` in the format:
17 
18 ```
19 username
20 password
21 ```

23 # ROS Services
24 
25 ## Fetch News
26 
27 Service URL: `/rapp/rapp_news_explorer/fetch_news`
28 
29 Service Type:
30 
31 NewsExplorerSrv.srv
32 ```
33 # Request
34 # The news search engine
35 string newsEngine
36 # Desired keywords
37 string[] keywords
38 # Reject list of previously read articles, in order to avoid duplicates
39 string[] excludeTitles
40 # Language/Region
41 string regionEdition
42 # Main topics, i.e. sports, politics etc
43 string topic
44 # Number of news stories
45 int8 storyNum
46 
47 ---
48 # Response
49 string error # Error description
50 
51 # List of news articles
52 rapp_platform_ros_communications/NewsStoryMsg[] stories
53 ```
54 **Available newsEngine values:**
55 * '' (uses default news engine)
56 * 'google'
57 * 'event_registry'
58 
59 NewsStoryMsg.msg
60 ```
61 # Article title
62 string title
63 # Article brief content
64 string content
65 # Article publisher
66 string publisher
67 # Article publication date
68 string publishedDate
69 # Article original url
70 string url
71 ```
