3 ## backup
4 
5 The following scripts are offered:
6 - ```dumpRAPPMysqlDatabase.sh```: Dumps the RAPP MySQL database in a .sql file
7 - ```importRAPPMysqlDatabase.sh```: Imports the RAPP MySQL database from a .sql file
8 - ```dumpRAPPOntology.sh```: Dumps the RAPP ontology in an .owl file
9 - ```importRAPPOntology.sh```: Imports the RAPP ontology from an .owl file

11 ## deploy
12 
13 There are two files aimed for deployment:
14 
15 - ```deploy_rapp_ros.sh```: Deploys the RAPP Platform back-end, i.e. all the ROS nodes
16 - ```deploy_hop_services.sh```: Deploys the corresponding HOP services
17 
18 Follow the next steps to deploy the RAPP Platform software:
19 
20 - ```screen```
21 - ```./deploy_rapp_ros.sh```
22 - Press Ctrl + a + d to detach
23 - ```screen```
24 - ```./deploy_hop_services```
25 - Press Ctrl + a + d to detach
26 - ```screen -ls``` to check that 2 screen sessions exist
27

29 ```
30 screen -r [pid.]tty.host

35 ## setup
36 
37 These scripts can be executed after a clean Ubuntu 14.04 installation, in order
38 to install the appropriate packages and setup the environment.
39 
40 #### Step 0 - Github installation
41 
42 Install git in order to be able to clone the repositories:
43 
44 ```sudo apt-get install git mercurial```
45 
46 WARNING: At least 10 GB's of free space are recommended.
47 
48 #### Install RAPP Platform
49 
50 It is advised to execute the clean_install.sh script in a clean VM or clean system.
51 
52 Performs:
53 - initial system updates
54 - installs ROS Indigo
55 - downloads all Github repositories needed
56 - builds and install all repos (rapp_platform, knowrob, rosjava)
57 - downloads builds and installs depending libraries for Sphinx4
58 - installs MySQL
59 - installs HOP
60 
61 #### What you must do manually
62 
63 A new MySQL user was created with username = ```dummyUser``` and password = ```changeMe``` and was granted all on RappStore DB. It is highly recommended that you change the password and the username of the user. The username and password are stored in the file located at /etc/db_credentials. The file db_credentials is used by the RAPP platform services, be sure to update it with the correct username and password. It's first line is the username and it's second line the password.
64 
65 #### Setup in an existing system
66 
67 If you want to add rapp-platform to an already existent system (Ubuntu 14.04) you should choose which setup scripts you need to execute. For example if you have MySQL install you do not need to execute ```8_mysql_install.sh```.
68 
69 #### Scripts
70 
71 - ```1_system_updates.sh```: Fetches the Ubuntu 14.04 updates and installs them
72 - ```2_ros_setup.sh```: Installs ROS Indigo
73 - ```3_auxiliary_packages_setup.sh```: Installs software from apt-get, necessary for the correct RAPP Platform deployment
74 - ```4_rosjava_setup.sh```: Fetches a number of GitHub repositories and compiles rosjava. This is a dependency of Knowrob.
75 - ```5_knowrob_setup.sh```: Fetches the Knowrob repository and builds it. Knowrob is the tool that deploys the RAPP ontology.
76 - ```6_rapp_platform_setup.sh```: Fetches the rapp-platform and rapp-api repositories and builds them
77 - ```7_sphinx_libraries.sh```: Fetches the Sphinx4 necessary libraries, compiles them and installs them, Sphinx4 is used for ASR (Automatic Speech Recognition).
78 - ```8_mysql_install.sh```: Installs MySQL
79 - ```9_create_rapp_mysql_db.sh```: Adds the RAPP MySQL empty database in mysql
80 - ```10_create_rapp_mysql_user.sh```: Creates a user to enable access the to database from the RAPP Platform code
81 - ```11_hop_setup.sh```: Installs HOP and Bigloo, the tools providing the RAPP Platform generic services.

83 ##### NOTES:
84 
85 The following notes concern the manual setup of rapp-platform (not the clean setup form our scripts):
86 
87 - To compile ```rapp_qr_detection``` you must install the ```libzbar``` library
88 - To compile ```rapp_knowrob_wrapper``` you must execute the following scripts:
89  - https://github.com/rapp-project/rapp-platform/blob/master/rapp_scripts/setup/4_rosjava_setup.sh
90  - https://github.com/rapp-project/rapp-platform/blob/master/rapp_scripts/setup/5_knowrob_setup.sh
91  - **If you don't want interaction with the ontology, add an empty ```CATKIN_IGNORE``` file in the ```rapp-platform/rapp_knowrob_wrapper/``` folder**

93 ## documentation
94 
95 Scripts to automatically create documentation from the RAPP Platform code, the services and this wiki, using Doxygen.
96 All documents can be bound in ```${HOME}/rapp_platform_files/documentation```.
97 
98 - `create_source_documentation.sh` : Creates the platform's source code documentation (cpp, h, py, java).
99 - `create_wiki_documentation.sh` : Creates the platform's GitHub Wiki documentation.
100 - `create_test_documentation.py` : Creates the platform's test source code documentation.
101 - `create_web_services_documentation.sh` : Creates the rapp_web_services documentation.
102 - `create_documentation.sh` : Creates all aforementioned documentations.
