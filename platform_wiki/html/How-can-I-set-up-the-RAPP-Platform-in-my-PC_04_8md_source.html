1 The setup scripts are located in ```rapp_scripts/setup```.
2 
3 These scripts can be executed after a clean Ubuntu 14.04 installation, in order
4 to install the appropriate packages and setup the environment.
5 
6 #### Step 0 - Github installation
7 
8 Install git in order to be able to clone the repositories:
9 
10 ```sudo apt-get install git mercurial```
11 
12 WARNING: At least 10 GB's of free space are recommended.
13 
14 #### Install RAPP Platform
15 
16 It is advised to execute the clean_install.sh script in a clean VM or clean system.
17 
18 Performs:
19 - initial system updates
20 - installs ROS Indigo
21 - downloads all Github repositories needed
22 - builds and install all repos (rapp_platform, knowrob, rosjava)
23 - downloads builds and installs depending libraries for Sphinx4
24 - installs MySQL
25 - installs HOP
26 
27 #### What you must do manually
28 
29 A new MySQL user was created with username = ```dummyUser``` and password = ```changeMe``` and was granted all on RappStore DB. It is highly recommended that you change the password and the username of the user. The username and password are stored in the file located at /etc/db_credentials. The file db_credentials is used by the RAPP platform services, be sure to update it with the correct username and password. It's first line is the username and it's second line the password.
30 
31 #### Setup in an existing system
32 
33 If you want to add rapp-platform to an already existent system (Ubuntu 14.04) you should choose which setup scripts you need to execute. For example if you have MySQL install you do not need to execute ```8_mysql_install.sh```.
34 
35 #### Scripts
36 
37 - ```1_system_updates.sh```: Fetches the Ubuntu 14.04 updates and installs them
38 - ```2_ros_setup.sh```: Installs ROS Indigo
39 - ```3_auxiliary_packages_setup.sh```: Installs software from apt-get, necessary for the correct RAPP Platform deployment
40 - ```4_rosjava_setup.sh```: Fetches a number of GitHub repositories and compiles rosjava. This is a dependency of Knowrob.
41 - ```5_knowrob_setup.sh```: Fetches the Knowrob repository and builds it. Knowrob is the tool that deploys the RAPP ontology.
42 - ```6_rapp_platform_setup.sh```: Fetches the rapp-platform and rapp-api repositories and builds them
43 - ```7_sphinx_libraries.sh```: Fetches the Sphinx4 necessary libraries, compiles them and installs them, Sphinx4 is used for ASR (Automatic Speech Recognition).
44 - ```8_mysql_install.sh```: Installs MySQL
45 - ```9_create_rapp_mysql_db.sh```: Adds the RAPP MySQL empty database in mysql
46 - ```10_create_rapp_mysql_user.sh```: Creates a user to enable access the to database from the RAPP Platform code
47 - ```11_hop_setup.sh```: Installs HOP and Bigloo, the tools providing the RAPP Platform generic services.
48 
49 ##### NOTES:
50 
51 The following notes concern the manual setup of rapp-platform (not the clean setup form our scripts):
52 
53 - To compile ```rapp_qr_detection``` you must install the ```libzbar``` library
54 - To compile ```rapp_knowrob_wrapper``` you must execute the following scripts:
55  - https://github.com/rapp-project/rapp-platform/blob/master/rapp_scripts/setup/4_rosjava_setup.sh
56  - https://github.com/rapp-project/rapp-platform/blob/master/rapp_scripts/setup/5_knowrob_setup.sh
57  - **If you don't want interaction with the ontology, add an empty ```CATKIN_IGNORE``` file in the ```rapp-platform/rapp_knowrob_wrapper/``` folder**
