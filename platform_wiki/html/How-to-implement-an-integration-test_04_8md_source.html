Go to the documentation of this file.
1 It is recommended to study on [Rapp-Testing-Tools](https://github.com/rapp-project/rapp-platform/wiki/RAPP-Testing-Tools) first.
2 
3 Basic documentation on "**Developing Integration Tests**" can be found [here](https://github.com/rapp-project/rapp-platform/tree/master/rapp_testing_tools).
4 
5 A more-in-depth information on how to write your first integration test is presented here.
6 
7 The [template_test.py](https://github.com/rapp-project/rapp-platform/blob/master/rapp_testing_tools/scripts/default_tests/template_test.py) will be used as a reference.
8 
9 Each integration test must have the following characteristics:
10 
11 - Each test class is written as a seperate python source file (.py).
12 - Each test inherits from `unittest.TestCase` class.
13 - The [Python RAPP Platform API](https://github.com/rapp-project/rapp-api/tree/master/python) is used to call RAPP Platform Services.
14 
15 
16 If you are not familiar with the Python unit testing framework (unittest), start reading from [there](https://docs.python.org/2.7/library/unittest.html) as the rapp_testing_tools is build ontop of it.
17 Also, basic knowledge of using the [python-rapp-platform-api](https://github.com/rapp-project/rapp-api/tree/master/python) is required.
18 
19 
20 So lets create a test class for testing the integration behaviour of the Face-Detection RAPP Platform service/functionality.
21 
22 
23 #### Step 1: Create the source file for the test class
24 
25 Copy the `template_test.py` file and name it `face_detection_tests.py` under the `default_tests` directory
26 
27 ```bash
28 $ cd ~/rapp_platform/rapp-platform-catkin-ws/src/rapp-platform/rapp_testing_tools/scripts/default_tests
29 $ cp template_test.py face_detection_tests.py
30 ```
31 
32 Rename the test class to `FaceDetectionTests`:
33 
34 ```python
35 class FaceDetectionTests(unittest.TestCase):
36 
37  def setUp(self):
38  self.ch = RappPlatformAPI()
39 
40  rospack = rospkg.RosPack()
41  self.pkgDir = rospack.get_path('rapp_testing_tools')
42 
43  def test_templateTest(self):
44  self.assertEqual(1, 1)
45 
46 ```
47 
48 Note that the `setUp()` method instantiates a RappPlatformAPI() object for calling RAPP Platform Services.

50 
51 #### Step 2: Implement a test case
52 
53 As the `FaceDetectionTests` class inherits from `unittest.TestCase`, individual tests are defined with methods whose names start with the letters `test`
54 
55 We will implement a test of performing face-detection on a single image file (Lenna.png). We assume that the image file was previously stored under the `test_data` directory. So lets create a member method named `test_lenna`
56 This method will be responsible for loading the image file, call the RAPP Platform Service, through the API, and evaluate the results using `unittest` assertions
57 
58 ```python
59 class FaceDetectionTests(unittest.TestCase):
60 
61  def setUp(self):
62  self.ch = RappPlatformAPI()
63 
64  rospack = rospkg.RosPack()
65  self.pkgDir = rospack.get_path('rapp_testing_tools')
66 
67  def test_lenna(self):
68  self.assertEqual(1, 1)
69  response = self.ch.faceDetection(imagepath)
70 
71  valid_faces = [{
72  'up_left_point': {'y': 201.0, 'x': 213.0},
73  'down_right_point': {'y': 378.0, 'x': 390.0}
74  }]
75 
76  self.assertEqual(response['error'], u'')
77  self.assertEqual(response['faces'], valid_faces)
78 ```
79 
80 The first assertion, `self.assertEqual(response['error'], u'')` evaluates that no error was reported from the RAPP Platform
81 The second assertion evaluates the response from the FaceDetector.
